HTML

完整的HTML结构

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Title</title>
    </head>
    <body>
    </body>
</html>
  • 文档声明（<!DOCTYPE html>）
    • 告诉浏览器当前页面为HTML5页面，让浏览器用HTML5的标准去解析HTML文档
    • 放在HTML文档的最前面，不能省略，否则会出现兼容性问题
    • HTML5的文档声明比HTML4.01、XHTML1.0简洁
  • html元素
    • 是HTML文档的根元素，一个文档中只能有一个，其他元素都是它的后代元素
    • W3C标准建议为html增加一个lang属性，作用是指定当前页面使用的语言，帮助翻译工具确定使用的翻译规则，帮助语音合成工具确定使用的（lang=”en”表示文档语言为英文，lang=”zh”表示文档语言为中文）
  • head元素（放一些元数据）
    • 元数据
      • 用来描述数据的数据
      • 对网页进行一些基本设置：字符编码、网页标题、网页图标
    • title元素
      • 设置网站的标题
    • meta元素
      • 设置网页的字符编码：不设置或设置错误会导致乱码
        <meta charset="UTF-8">
    • style元素：样式
    • link元素：引入外部css文件、设置网站图标 
      // stylesheet表示引入样式
<link rel="stylesheet" href="">
// icon表示引入图标 type可不写
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
    • base元素
    • script元素
    • noscript元素
  • body元素（包含网页的具体内容和结构）

HTML基本元素

  • h元素

    • 表示网页的标题
    • h1~h6共规定了6个等级的标题
    • 演示
      <h1>hello world</h1>
<h2>hello world</h2>
<h3>hello world</h3>
<h4>hello world</h4>
<h5>hello world</h5>
<h6>hello world</h6>

  • p元素

    • 表示段落
    • 演示
      <p>我在这头，你在那头</p>

  • strong元素

    • 表示加粗、强调效果
    • 演示
      <p>天行健，君子以<strong>自强不息</strong>，地势坤，君子以<strong>厚德载物</strong></p>

  • h元素和SEO优化（搜索引擎优化）

    • SEO优化（百度、搜狗、360、Google搜索引擎）
    • h元素有助于网站的SEO优化，可以促进关键词排名
    • 建议在网页中最多只有一个h1元素

  • code元素（代码元素）

    • 特点：将字体设置为等宽字体

  • br元素（换行）

  • hr元素（分割线）

  • 字符实体

    • 小于号（<）：&lt;
    • 大于号（>）：&gt;
    • 空格：&nbsp;
    • 双引号（”）：&quot;
    • 单引号（’）：&apos;

  • span元素

    • 演示
      <span>我是span标签</span>

  • div元素

    • 演示
      <div>
  <h2>hello world</h2>
  <p>p标签</p>
</div>

  • img标签

    • 该标签为单标签
    • 常见图片格式
      • png：静态图片，支持透明
      • jpg（jpeg）：静态图片，不支持透明
      • gif：动态图片，支持透明
      • webp
    • src属性：指定图片路径（绝对路径、相对路径）
    • alt属性：当图片失效（加载失败）时显示的文本
    • width属性：设置图片宽度

  • a元素

    • 用于定义超链接，打开新的URL

    • href属性：指定要打开的URL

    • target属性

      • _self（默认值）：在当前页面打开
      • _blank：在新的页面打开
      • _parent：需要和iframe一起使用才具有效果
      • _top：需要和iframe一起使用才具有效果
      • 具体的name：需要和iframe一起使用才具有效果

    • 演示 

      <div id="top">顶端内容</div>
<a href="https://www.baidu.com" target="_blank">百度</a>

// 锚点链接 通过id到达指定位置
<a href="#top">回到顶端</a>

// 伪链接 不指定具体跳转地址，暂时不做任何事情
<a href="#" onclick="return false;">伪链接一</a>
<a href="javascript:;">伪链接二</a>

  • a元素和base元素结合使用

    • 演示
      <head>
  <!--抽取-->
  <base href="https://www.baidu.com" target="_blank">
</head>
<body>
  <a href="">百度一下</a>
  <a href="/s?wd=vue">搜索</a>
  <!--不会拼接-->
  <a href="https://www.taobao.com">淘宝一下</a>
</body>

  • iframe元素

    • 演示
      <iframe 
    src="https://www.baidu.com" 
    width="1000" 
    height="500" 
    frameborder="1">
</iframe>

  • URL（统一资源定位符 ）

    • 资源的地址、位置，具有唯一性
    • 协议://主机地址（域名或IP地址）:端口/路径?查询#锚点位置

  • 标签语义化（使用最适合的标签做最适合的事情）

    • 方便代码维护
    • 减少开发者之间的沟通成本
    • 让搜索引擎正确识别重要信息

CSS（层叠样式表）

CSS引入的三种方式

  • 内联样式（inline style）
    <h1 style=
"font-size: 50px;color: red">网页的标题</h1>
  • 文档样式表（document style sheet）、内嵌样式表（embed style sheet）
    <style>
  h1 {
    color: red;
    font-size: 40px;
  }
</style>
  • 外部样式表（external style sheet）
    // 外部引入样式文件
<link rel="stylesheet" href="style.css">

CSS编码和import引入

  • CSS编码
    // 给外部css文件指定编码
@charset "utf-8";
  • import引入外部css文件
    <style>
  // 可以引入多个外部css文件 比之link引入效率较低
  @import url("外部css路径");
  color: blue;
</style>

CSS选择器

  • 通配选择器（通配符，遍历）

    <style>
  * {
    margin: 0;
    padding: 0;
    font-size: 12px;
  }
</style>

  • 标签选择器

    <style>
  div {
    width: 200px;
    height: 100px;
  }
</style>

  • 类选择器

    <style>
  .box {
    background-color: red;
  }
</style>

  • id选择器

    #box {
  background-color: blue;
}

  • 属性选择器

    <div title="one div">我是div</div>
<p title="one p">我是p</p>
<style>
  /*
  [title="div元素"] {
    color: red;
  }
  */
  // 只要包含one
  [title*="one"] {
    color: red;
  }
  // 以one开头
  [title^="one"] {
    color: red;
  }
  // 以one结尾
  [title$="one"] {
    color: red;
  }
</style>

  • 后代选择器

    <div>
  <span>1</span>
  <p>
    <span>2</span>
  </p>
</div>
<style>
  div span {
    color: red; 
  }
</style>
// 结果是 1 2变红
<style>
  div > span {
    color: red;
  }
</style>
// 结果是 1变红

  • 兄弟选择器

    // 后面紧挨着的元素
<span>1</span>
<div>
  <span>2</span>
<div>
<span>3</span>
<span>4</span>

<style>
  div+span {
    color: red;
  }
</style>
// 只有 3变红

<style>
  div~span {
    color: red;
  }
</style>
// 3 4变红

  • 交集选择器和并集选择器

    <div class="box" title="test"></div>
<div id="content"></div>
<style>
  // 交集选择器
  div.box[title="test"] {
    width: 200px;
    height: 200px;
  }
  .box, #content {
    color: red;
  }
</style>

  • 伪类选择器

    • 动态伪类
      <a href="#">链接</a>
<style>
  // 未访问的链接
  a:link {
    color: red;
  }
  // 已访问的链接
  a:visited {
    color: blue;
  }
  // 获取焦点
  a:focus {
    background-color: red;
  }
  // 鼠标挪到链接 必须放在link和visited的后面才会生效
  a:hover {
    color: green;
  }
  // 鼠标在链接上按住未松手 必须放在hover后面才能生效
  a:active {
    color: yellow;
  } 
</style>
    • 目标伪类
      <a href="#title">标题</a>
<h2 id="title">标题</h2>
<style>
  :target {
    color: red;
  }
</style>
    • 语言伪类
    • 元素状态伪类
      <button disabled>按钮</button>
<button>按钮</button>
<style>
  :disabled {
    color: red;
  }
</style>
    • 结构伪类
      <div class="box">
  <p>1</p>
  <p>2</p>
  <p>3</p>
  <p>4</p>
</div>
<style>
  // 2变成红色
  .box:nth-child(2) {
    color: red;
  }
  // 所有偶数变红 even
  .box:nth-child(2n) {
    color: red;
  }
  // 所有奇数变红 odd
  .box:nth-child(2n+1) {
    color: red;
  }
  // 倒数第一个元素变红
  .box:nth-last-child(1) {
    color: red;
  }
  .box:nth-of-type(1) {
    color: blue;
  }
</style>
    • 否定伪类
      <div>div元素</div>
<p>p元素</p>
<p>p元素</p>
<span>span元素</span>
<style>
  body :not(div) {
    color: red;
  }
</style>

文字属性

  • text-decoration
    • none 无下划线
    • underline 下划线
    • overline 上划线
    • line-through 删除线
  • letter-spacing 字母间间距
  • word-spacing 单词间间距
  • text-transform 大小写转换
    • capitalize 首字母大写
    • uppercase 大写
    • lowercase 小写
    • none 无
  • text-indent 设置第一行缩进
  • text-align 设置元素内容在元素中水平对齐方式
    • left 左对齐
    • right 右对齐
    • center 居中显示
    • justify 左右两端对齐（对最后一行无效果）
  • font-size 设置字体大小
  • font-family 设置字体名称
    p {
  // 可设置多个字体 
  font-family: '微软雅黑',arial;
}
  • font-weight 设置文字的粗细（可以是具体值）
    • bold 加粗 相当于700
    • normal 常规 相当于400
  • font-style 设置文字的样式
    • normal 常规
    • italic 斜体（font-family这种字体支持斜体）
    • oblique 斜体（让文字倾斜）
  • font-varient
  • line-height 设置行高（两行文字基线baseline之间的距离）
